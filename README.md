# Web Contact Management

Aplikasi manajemen kontak berbasis Website dengan tampilan sederhana memiliki fitur:

[x] Menyimpan kontak
[ ] Menampilkan kontak
[ ] Menampilkan detail kontak
[ ] Mengedit kontak
[ ] Menghapus kontak
[ ] Pencarian kontak

Aplikasi ini dibuat menggunakan:

**Front End**
- HTML
- CSS
- JavaScript

**Back End**
- PHP
- SQLite / MySQL
